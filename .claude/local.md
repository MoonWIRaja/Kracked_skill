# KRACKED Skill - Local Configuration

This file contains local configuration overrides for the KRACKED plugin.

## Settings

| Setting | Value | Description |
|---------|--------|-------------|
| defaultLanguage | EN | Default language (EN or MS) |
| scale | standard | Project scale (small, standard, deep) |

## Override these values for your local environment

```yaml
defaultLanguage: EN  # or MS for Bahasa Melayu
scale: standard
```

## Notes

- Changes here affect only your local installation
- Team members should have their own local configs
- Do not commit this file to version control
