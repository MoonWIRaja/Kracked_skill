# Changelog

All notable changes to KRACKED_skill will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-13

### Added
- Initial release of KRACKED_skill (KD)
- Full KRACKED v3.3 system prompt
- 9 structured roles (Analyst, PM, Architect, Tech Lead, Engineer, QA, Security, DevOps, Release Manager)
- 7 workflow stages (Discovery, Requirements, Architecture, Implementation, Quality, Deployment, Release)
- Multi-agent support: Party Mode and Agent Swarm
- Confidence scoring and conflict resolution
- Dual language support (English & Bahasa Melayu)
- Persistent status tracking via `status.md`
- Decision validation blocks with structured checkpoints
- Adapters for Claude Code, Cursor, and Antigravity
- Generic adapter for other AI tools
- Installation system via `curl | bash` or `wget | bash`
- 9 document templates
- 6 quality checklists
- 4 workflow definitions
- Complete documentation suite
- Example projects
- Test suite
